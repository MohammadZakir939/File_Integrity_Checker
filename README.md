# File_Integrity_Checker

A security-focused Python tool that monitors the integrity of log files by computing and storing SHA-256 hash fingerprints. On first run, it establishes a trusted baseline. On every subsequent run, it recomputes hashes and compares them — flagging any file that was modified, even by a single character. Supports all files
